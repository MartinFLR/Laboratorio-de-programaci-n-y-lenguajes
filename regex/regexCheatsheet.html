Tabla de Expresiones Regulares (Regex)

Símbolo / Sintaxis Significado Ejemplo Qué hace
. Cualquier carácter excepto salto de línea a.b Coincide con "acb", "arb", etc.
* 0 o más repeticiones lo* "l", "lo", "loo", "looo"
+ 1 o más repeticiones lo+ "lo", "loo", "looo" (pero no solo "l")
? 0 o 1 vez colou?r Coincide con "color" o "colour"
[] Cualquier carácter dentro del conjunto [aeiou] Coincide con cualquier vocal
[^] Negación: cualquier carácter que no esté dentro [^aeiou] Coincide con consonantes
^ Inicio de línea ^Hola Coincide con frases que empiezan con "Hola"
$ Fin de línea mundo$ Coincide con frases que terminan en "mundo"
{n} Exactamente n repeticiones a{3} Coincide con "aaa"
{n,m} Entre n y m repeticiones a{2,4} "aa", "aaa", o "aaaa"
\w Letra, número o guión bajo (equivale a [A-Za-z0-9_]) \w+ Coincide con una o más letras o números
\d Cualquier dígito (0-9) \d{2} Coincide con dos números seguidos
\s Espacio en blanco hola\s+mundo Coincide con "hola mundo" (uno o más espacios entre)
\b Borde de palabra \bsol\b Coincide solo con "sol", no con "solución"
\ Escape de caracteres especiales \. Coincide con el punto "." literal
| Alternativa (o) gato|perro Coincide con "gato" o "perro"
( ) Agrupación (abc)+ Coincide con "abc", "abcabc", etc.
/.../g Bandera global "ana".match(/a/g) Busca **todas** las ocurrencias, devuelve ["a", "a"]
/.../i Bandera insensitive (sin distinguir mayúsculas/minúsculas) "Ana".match(/a/i) Coincide con "A" o "a"
/.../gi Global + insensitive "Ana ama a Ada".match(/a/gi) Devuelve todas las "a" o "A" encontradas: ["A", "a", "a", "A", "a"]
match() Método de cadena para buscar con regex frase.match(/a/gi) Devuelve un array con todas las coincidencias
(... || []) Operador OR: evita errores si match da null (frase.match(/a/gi) || []).length Cuenta cuántas veces aparece "a", aún si no hay ninguna
/.../g Bandera global "ana".match(/a/g) Devuelve todas las coincidencias: ["a", "a"]
/.../i Ignorar mayúsculas/minúsculas "Ana".match(/a/i) Coincide con "A" o "a"
/.../gi Global + case-insensitive "Ana ama a Ada".match(/a/gi) Devuelve todas las "a" o "A": ["A", "a", "a", "A", "a"]
match() Método que busca coincidencias frase.match(/a/gi) Devuelve un array de coincidencias o null
test() Prueba si hay coincidencias /hola/.test("hola mundo") Devuelve true o false
replace() Reemplaza texto "hola".replace(/o/, "a") Devuelve "hala"
search() Devuelve la posición de la primera coincidencia "hola mundo".search(/m/) Devuelve 5
\W No alfanumérico \W+ Coincide con signos y espacios
\D No dígito \D+ Coincide con letras, símbolos, etc.
\S No espacio \S+ Coincide con secuencias sin espacios
(?=...) Lookahead positivo \d(?=px) Coincide con un dígito seguido de "px", sin incluir "px"
(?!...) Lookahead negativo \d(?!px) Coincide con un dígito que **no** va seguido de "px"
(?<=...) Lookbehind positivo (?<=\$)\d+ Coincide con un número que viene después de "$"
(?<!...) Lookbehind negativo (?<!\$)\d+ Coincide con número **no** precedido por "$"
(?:...) Agrupación sin captura (?:abc)+ Repite "abc" pero no captura para backreferences
\n, \r, \t Saltos de línea, retorno de carro, tabulación "linea1\nlinea2" Coinciden con saltos reales en el texto
